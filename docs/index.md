# PythonProjects Documentation

Welcome to the PythonProjects documentation.

## Overview

This repository contains various Python projects for learning and reference purposes.

## Projects

The following projects are available in this repository:

- **blackjack** - A simple blackjack card game
- **bubble_sort** - Implementation of bubble sort algorithm
- **caesar_cipher** - Encrypt and decrypt messages using Caesar cipher
- **calculator** - A basic calculator with multiple operations
- **change_calc** - Calculate coin change for a given amount
- **check_palindrome** - Check if a string is a palindrome
- **coin_estimator** - Estimate coin values
- **coin_flip** - Simulate coin flips
- **collatz** - Collatz conjecture implementation
- **count_vowels** - Count vowels in a string
- **descriptive_stats** - Calculate descriptive statistics
- **dice_roller** - Roll dice of various sizes
- **factor_calc** - Calculate factors of numbers
- **fibonacci** - Fibonacci sequence generator
- **fizz_buzz** - Classic FizzBuzz implementation
- **gregory_leibniz** - Pi calculation using Gregory-Leibniz series
- **higher_lower** - A number guessing game
- **magic_eight_ball** - Magic 8-ball fortune teller
- **multiplication_table** - Generate multiplication tables
- **nilakantha** - Pi calculation using Nilakantha series
- **ninety_nine_bottles** - The 99 bottles of beer song
- **pig_latin** - Convert text to Pig Latin
- **py_triple** - Pythagorean triple finder
- **reverse_string** - Reverse a string
- **rock_paper_scissors** - Rock, paper, scissors game
- **sieve_of_eratosthenes** - Prime number sieve
- **tic_tac_toe** - Tic-tac-toe game
- **vigenere_cipher** - Vigen√®re cipher encryption
