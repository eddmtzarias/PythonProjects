# Python Projects Documentation

Welcome to the Python Projects documentation!

## Overview

This repository contains various Python projects for learning and practice, including:

- **Algorithms**: Sorting, searching, and mathematical algorithms
- **Games**: Blackjack, Tic-Tac-Toe, Rock Paper Scissors
- **Utilities**: Calculators, converters, and string manipulation tools

## Projects

| Project | Description |
|---------|-------------|
| `blackjack.py` | A simple blackjack card game |
| `bubble_sort.py` | Bubble sort algorithm implementation |
| `caesar_cipher.py` | Caesar cipher encryption/decryption |
| `calculator.py` | Basic calculator |
| `fibonacci.py` | Fibonacci sequence generator |
| `fizz_buzz.py` | Classic FizzBuzz program |
| `tic_tac_toe.py` | Tic-Tac-Toe game |
| And more... | See the `projects/` directory |

## Getting Started

### Prerequisites

- Python 3.10 or higher
- [Bazel](https://bazel.build) (for building)

### Running Projects

```bash
# Using Bazel
bazel run projects:{project_name}

# Example
bazel run projects:magic_eight_ball
```
