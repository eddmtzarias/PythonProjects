# Python Projects

Welcome to the Python Projects documentation.

## Overview

This repository contains various Python projects built with [Bazel](https://bazel.build).

## Available Projects

The following projects are available in this repository:

- **blackjack** - A blackjack card game
- **bubble_sort** - Bubble sort algorithm implementation
- **caesar_cipher** - Caesar cipher encryption/decryption
- **calculator** - A simple calculator
- **change_calc** - Change calculator
- **check_palindrome** - Palindrome checker
- **coin_estimator** - Coin value estimator
- **coin_flip** - Coin flip simulator
- **collatz** - Collatz conjecture calculator
- **count_vowels** - Vowel counter
- **descriptive_stats** - Descriptive statistics calculator
- **dice_roller** - Dice rolling simulator
- **factor_calc** - Factor calculator
- **fibonacci** - Fibonacci sequence generator
- **fizz_buzz** - FizzBuzz implementation
- **gregory_leibniz** - Gregory-Leibniz series for Pi
- **higher_lower** - Higher or lower guessing game
- **magic_eight_ball** - Magic 8-ball simulator
- **multiplication_table** - Multiplication table generator
- **nilakantha** - Nilakantha series for Pi
- **ninety_nine_bottles** - 99 bottles of beer song
- **pig_latin** - Pig Latin translator
- **py_triple** - Pythagorean triple finder
- **reverse_string** - String reverser
- **rock_paper_scissors** - Rock, paper, scissors game
- **sieve_of_eratosthenes** - Sieve of Eratosthenes prime finder
- **tic_tac_toe** - Tic-tac-toe game
