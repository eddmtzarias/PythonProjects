## Quick Start

```bash
# Clone the repository
git clone https://github.com/eddmtzarias/PythonProjects.git
cd PythonProjects

# Install development dependencies
pip install -r requirements-dev.txt

# Run a project
python projects/magic_eight_ball.py
```
